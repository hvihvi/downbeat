# Downbeat

Band rehearsal scheduling app — helps bandmates indicate their availability and find common times to rehearse.

## Tech stack
- Expo SDK 54, React Native 0.81, React 19
- TypeScript, Expo Router (file-based routing in `app/`)
- ESLint via `eslint-config-expo` (flat config)

## Commands
- `npm start` — start dev server
- `npm run lint` — run ESLint
